#  Algorithm Performance Analyzer

A Python-based project that benchmarks and compares the performance of classic sorting algorithms using empirical time complexity analysis. The project demonstrates how algorithm efficiency changes with increasing input sizes and visualizes performance differences using graphs.



# Overview

This project implements and compares multiple sorting algorithms by measuring their execution time across different dataset sizes. It provides a practical understanding of algorithm efficiency beyond theoretical Big-O notation by visualizing real-world performance behavior.



# Key Concepts

- Time Complexity (Big-O notation)
- Empirical performance analysis
- Algorithm optimization
- Divide-and-conquer strategy (Merge Sort)
- Data visualization using Matplotlib



# Features

- Implements multiple sorting algorithms:
  - Bubble Sort (O(n²))
  - Merge Sort (O(n log n))
- Measures execution time for varying input sizes
- Compares algorithm performance side-by-side
- Generates performance graphs using Matplotlib
- Saves visual results for documentation and analysis
